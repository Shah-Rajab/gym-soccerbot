# gym-soccerbot
Pybullet Gym Envrionment for Humanoid Soccerbot



Installation
```shell script
# Make sure you clone this repository in an empty directory
cd ..
sudo apt-get install python3-tk virtualenv
virtualenv -p python3 venv
. venv/bin/activate
cd gym-soccerbot
pip install -r req.txt

python3 main.py
```
